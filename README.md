# Statistical Analysis with Python

## Overview

This repository demonstrates hands-on implementation of numerical computing and statistical analysis workflows using Python libraries including **NumPy, SciPy, Pandas, and StatsModels**.

It covers foundational concepts ranging from array manipulation to inferential statistical testing and data-driven decision analysis.

---

## Topics Covered

### Numerical Computing

* N-dimensional array creation
* Indexing and slicing
* Matrix operations
* Random sampling
* Array reshaping and stacking

### Probability & Descriptive Statistics

* Central tendency measures
* Distribution analysis
* Variance and standard deviation
* Correlation analysis
* Simulation of probability distributions

### Inferential Statistics

* Z-tests
* T-tests
* Two-sample testing
* ANOVA
* Chi-Square testing
* Hypothesis interpretation using p-values
